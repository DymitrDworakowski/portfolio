## <Trans i18nKey="opis">Project Description</Trans>
**Contact Management Web Application** - This web application is designed to help users efficiently manage and organize their contacts. Built using React for the frontend, it offers a user-friendly interface to add, edit, delete, and search for contacts. The backend is developed with Node.js and is connected to a MongoDB database for secure and efficient data storage. The backend is deployed on Render.com to ensure reliable performance.

## <Trans i18nKey="features">Key Features</Trans>

- **<Trans i18nKey="features_1">Contact Management:</Trans>** Users can add, edit, delete, and view detailed information about their contacts, including name, phone number, and email.
- **<Trans i18nKey="features_2">Favorites Feature:</Trans>** Mark contacts as "favorites" to easily access frequently used contacts.
- **<Trans i18nKey="features_3">Responsive Design:</Trans>** The application is optimized for both desktop and mobile devices, allowing for seamless use across different screen sizes.
- **<Trans i18nKey="features_4">Real-Time Updates:</Trans>** Changes made to contacts are immediately reflected in the UI, providing a dynamic and responsive experience.
- **<Trans i18nKey="features_5">Authentication:</Trans>** User authentication is implemented to secure contact information, ensuring that only authorized users have access to their data.

## <Trans i18nKey="technologies">Technologies Used:</Trans>

- **<Trans i18nKey="technologies_1">Frontend:</Trans>** React.js, CSS for styling and responsive design
- **<Trans i18nKey="technologies_2">Backend:</Trans>** Node.js, Express.js for API management
- **<Trans i18nKey="technologies_3">Database:</Trans>** MongoDB for scalable data storage
- **<Trans i18nKey="technologies_4">Hosting:</Trans>** Backend is hosted on Render.com, while the frontend can be deployed on GitHub Pages or another static hosting service.

## <Trans i18nKey="how_it_works">How It Works:</Trans>

<Trans i18nKey="how_it_works_1">Users log in to the application.
They can add new contacts, view the list of existing contacts, mark important contacts as favorites, and delete or update contact details as needed.
The application automatically saves all changes to MongoDB, ensuring that contact data is always up-to-date.
This contact management application is ideal for individuals or small teams looking for a simple, reliable way to organize contact information.</Trans>
